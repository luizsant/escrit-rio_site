# Site Institucional - Luiz Santiago Advocacia

Projeto institucional em Vite, com identidade visual baseada na landing em `referencia` e estrutura inspirada no site de referencia informado.

## Desenvolvimento

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy na Vercel

1. Importar a pasta `institucional` no projeto da Vercel.
2. Framework preset: `Vite`.
3. Build command: `npm run build`.
4. Output directory: `dist`.
